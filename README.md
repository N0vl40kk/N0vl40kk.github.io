**VISUAL STORIES**

Официальное портфолио NXXSHOT.

Это иммерсивный веб-сайт, созданный для демонстрации визуальных историй.
Проект представляет собой интерактивную "стену" из 48 видеороликов с уникальной физикой движения, погружающую зрителя в атмосферу работ автора.

* **Направление:** Video Production / Visual Arts / Post Production
* **Стиль:** Cinematic / Dark / Immersive
